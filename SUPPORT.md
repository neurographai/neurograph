# Support

## How to get help

This project uses GitHub Issues and Discussions for support.

- **Bugs**: [Open an issue](https://github.com/neurographai/neurograph/issues/new?template=bug_report.yml)
- **Feature requests**: [Open a feature request](https://github.com/neurographai/neurograph/issues/new?template=feature_request.yml)
- **Questions**: [Start a discussion](https://github.com/neurographai/neurograph/discussions)
- **Security issues**: See [SECURITY.md](SECURITY.md) for responsible disclosure

## Response times

This is an open-source project maintained by volunteers. We do our best to respond to issues and discussions in a timely manner, but there are no guaranteed response times.
