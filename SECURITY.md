# Security Policy

## Supported Versions

Currently, we only support the main branch and the latest stable release for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |
| < v0.1  | :x:                |

## Reporting a Vulnerability

Please do NOT report security vulnerabilities via public GitHub issues.

Instead, please report them to security@neurographai.com or reach out privately to the maintainers on Discord/Email. We will acknowledge your report within 48 hours and provide a timeline for providing a fix.
